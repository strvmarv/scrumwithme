# Changelog

## 2026-02-24

### Added
- **Hours and Days estimation methods** — time-based estimation options alongside existing methods
- **Pass/abstain vote** — voters can pass on a ticket without blocking the session
- **Vote timer with auto-pass** — configurable countdown that auto-passes AFK voters when time expires
- **Copyable session link** — host can copy the join URL to share via chat
- **Username validation hint** — guidance when name is too short to join

### Changed
- Kick button restyled for better visibility
